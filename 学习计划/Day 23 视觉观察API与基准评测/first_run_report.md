# Day 23 首轮视觉基准记录

日期：2026-07-23

## 范围

本次只评测一张由用户主动选择、人工标注且未上传仓库的 Minecraft 截图。图片通过本地 SSH 隧道发送至 Day 23 的只读观察 API；API 调用独立的视觉 vLLM 服务，再将结果交给 Day 22 观察守卫校验。

该测试不向命令网关传递图片，也不会生成或执行 Mindcraft 动作。

## 结果

- 样本数：`1`
- 守卫接受样本：`1/1`
- 必需标签覆盖率：`61.54%`
- 场景类必需标签：`4/4` 命中
- 方块类必需标签：`4/9` 命中

## 解读

端到端链路已验证：本地图片编码、SSH 隧道、`POST /observe`、视觉 vLLM、JSON 观察守卫和本地基准脚本均成功工作。结果也暴露出视觉模型会遗漏远处、细小或材质细节较弱的方块，因此观察输出只能作为只读线索，不能直接转化为游戏动作。

单个样本不足以估计模型整体能力。下一轮基准应加入不同距离、光照、天气、方块密度和实体数量的截图，并继续保留独立人工标注与私有报告。
